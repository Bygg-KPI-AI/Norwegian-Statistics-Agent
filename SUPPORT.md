# Support

Support is provided on a best-effort basis via GitHub Issues in this repository. There is no SLA and no warranty.

## How to open an issue
- Go to the repository’s Issues tab.
- Create a new issue with a clear title and description.
- Include:
  - What you were trying to do (search, parameter lookup, data import).
  - The SSB table ID (if applicable) and any parameter filters you used.
  - Screenshots or the exact response/error message, if available.
  - Steps to reproduce, if the issue is consistent.

## Scope of support
- Guidance on using the agent’s features.
- Troubleshooting common errors and data size/limit issues.
- Best-effort updates; availability and maintenance are not guaranteed.

## Not covered
- Guarantees of availability, performance, or completeness of datasets.
- Custom data transformation, dashboards, or long-term maintenance.
