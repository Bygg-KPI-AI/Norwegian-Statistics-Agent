# Privacy

This agent does not collect personal data.

- The agent’s purpose is to query Statistics Norway (SSB) PX-Web endpoints and present results inside Microsoft 365 Copilot Chat.
- The agent itself does not retain conversation content or user identifiers.
- Platform-level logging or retention may apply in Microsoft Teams/Microsoft 365 Copilot; please refer to Microsoft’s service and privacy policies for details.
- No persistent storage by the agent is used for imported data; results are presented within the chat.

If you have questions about privacy or data handling, please open an issue in this repository.
